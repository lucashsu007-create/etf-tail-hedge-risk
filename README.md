# ETF Tail Hedge Risk

Research question:

Do near-substitute ETFs experience extreme intraday hedge-error events during liquidity stress, and are these events underestimated by rolling beta/correlation models?

First universe:
- SPY / IVV / VOO
- QQQ / QQQM

First data source:
- WRDS TAQ trades and quotes once access is approved

First empirical object:
HE_t = r_ETF1,t - beta_t r_ETF2,t
